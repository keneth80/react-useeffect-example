## 케이스 주제

Q. react의 useEffect함수를 이해하기 위해 구현을 해보자.


## 기능 요구사항

1. effect를 실행하는 함수와 의존성 데이터를 저장한다.


2. html element를 렌더링 후에 effect를 실행할 수 있도록 한다.


3. 의존성 데이터를 비교하여 effect 실행여부를 결정한다.


4. effect 함수를 통해 document의 title을 변경한다.
